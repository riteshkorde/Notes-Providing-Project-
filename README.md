### Run Locally

To run **EduHome** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/eduhome.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/eduhome.git
```
